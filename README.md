## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Installations
* npm
  ```sh
  npm install npm@latest -g
  ```


### Installation

1. Clone the Fretsy
   ```sh
   git clone https://github.com/gch910/Fretsy.git


2. Install NPM packages - run in frontend and backend directories
   ```sh
   npm install
   ```
